• 'CTRL+SHIFT+v': increase Font Size
  • 'CTRL+SHIFT+l': decrease Font Size

  • 'CTRL+SHIFT+e': increase Row Size
  • 'CTRL+SHIFT+h': decrease Row Size
    Worst case: no picture, extra context, row highlighting, transition or animation.
    'CTRL+SHIFT+s': Fast case with all options; picture, extra context, row highlighting, transition, and animation.
